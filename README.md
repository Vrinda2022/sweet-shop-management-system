# Sweet Shop Management System

## Tech Stack
- Backend: Node.js, TypeScript, Express
- Database: MongoDB Atlas
- Frontend: React + TypeScript
- Auth: JWT
- Testing: Jest, Supertest

## Setup
1. Backend
   - cd backend
   - npm install
   - npm run dev

2. Frontend
   - cd frontend
   - npm install
   - npm start
